## Projects

**EDA with LLM — Ollama + DeepSeek** &nbsp;·&nbsp; `Python` `Ollama` `DeepSeek` `LangChain`

Built a local LLM pipeline that ingests any tabular dataset and auto-generates structured insights, reducing manual exploratory analysis time by 60%. The system uses DeepSeek running via Ollama with a custom prompt chain for column-level and cross-feature reasoning.

---

**IIPL Data Analysis & Visualization** &nbsp;·&nbsp; `Python` `Pandas` `Seaborn`

Analysed 10+ years of IPL match data to surface team performance trends, toss impact, and winning-condition patterns. Built a visual report with annotated charts highlighting key season-level shifts.

---

**IMDB Movie Insights Engine** &nbsp;·&nbsp; `Python` `EDA` `Matplotlib`

Cleaned and explored a 5,000+ film dataset to identify which genres yield the highest ROI against production budget. Surfaced findings about mid-budget genre films outperforming blockbusters on return.

---

**Global GDP Trend Analysis** &nbsp;·&nbsp; `Python` `Pandas` `Plotly`

Multi-decade time-series analysis across 150+ countries, handling 30%+ missing data through interpolation. Built animated regional growth charts showing economic power shifts from 1980 to present.
